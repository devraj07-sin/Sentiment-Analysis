# Sentiment Analysis using NLP & Machine Learning

## 📌 Project Overview
This project focuses on analyzing text reviews and classifying them into different sentiments:

- Positive 😀
- Negative 😡
- Neutral 😐

The project uses Natural Language Processing (NLP) techniques and Machine Learning for text classification.

---

## 🎯 Problem Statement
Understanding customer opinions from reviews is important for businesses.

This project helps identify whether a review expresses:
- Positive feedback
- Negative feedback
- Neutral opinion

---

## 📂 Dataset
Custom sentiment dataset containing text reviews and sentiment labels.

### Features:
- Review
- Sentiment

---

## 🛠️ Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib
- Seaborn

---

## ⚙️ Workflow

### 1. Data Loading
- Loaded CSV dataset using Pandas

### 2. Text Preprocessing
- Lowercasing
- Removing special characters
- Tokenization
- Stopword removal

### 3. Feature Extraction
- TF-IDF Vectorization

### 4. Model Training
- Naive Bayes Classifier

### 5. Model Evaluation
- Accuracy Score
- Classification Report

---

## 🤖 Machine Learning Model
- Multinomial Naive Bayes

---

## 📊 NLP Techniques Used

The model successfully classified text reviews into:
- Positive
